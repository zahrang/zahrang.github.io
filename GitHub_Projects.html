Tic-Tac-Toe Game Theory Analysis

A simulation-based game theory study analysing how optimal vs imperfect strategies affect outcomes in Tic-Tac-Toe.

Python • Game Theory • Simulation • Probability • Statistical Analysis

This project models thousands of Tic-Tac-Toe games under different player skill levels to understand how deviations from optimal play affect win/draw/loss distributions. It demonstrates that while perfect play guarantees a draw, introducing realistic human error shifts outcomes significantly.

Key analysis includes: perfect vs perfect play (theoretical draw equilibrium), perfect vs imperfect strategy dominance, imperfect vs imperfect balance effects, and random baseline behaviour.
