Stock Portfolio Tracker

Python • Data Analysis • Visualisation

Overview

This project is a Python-based portfolio analysis tool that evaluates stock performance, computes returns, and generates visualisations to better understand portfolio behaviour.

Data

The analysis focuses on three companies:

  • TSLA — Tesla, Inc.
  • AAPL — Apple Inc.
  • MSFT — Microsoft Corporation

Process

  • Collected historical stock price data
  • Calculated daily returns and cumulative returns
  • Analysed relationships between assets
  • Generated visual outputs to interpret performance

Visualisations

Historical price movement of selected stocks

Cumulative portfolio performance over time

Correlation between assets in the portfolio

Key Insight

The results show how diversification impacts overall volatility and returns, highlighting the importance of correlation when constructing a portfolio.
